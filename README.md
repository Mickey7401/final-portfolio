# Michael Pereira - Web Development Portfolio

Hi, I’m Michael Pereira, a Digital Media student at UCF focused on web development, UI/UX design, and AI-assisted development. This portfolio highlights two projects I worked on during DIG4503C: my Movie Watchlist App and my Lakeside Coffee Co. static website.

## Featured Projects

---

## Movie Watchlist App

The Movie Watchlist App is a web application where users can search for movies, save favorites, and manage a personal watchlist. This was my main project for the semester and helped me practice building a more complete web app with real user interaction and external data.

**Tech Stack:**  
React, JavaScript, CSS, Firebase, TMDb API, GitHub, Deployment

**Project Link:**  
https://michael-movie-watchlist.netlify.app

**GitHub Repository:**  
https://github.com/Mickey7401/movie-watchlist

**What I Learned:**  
This project helped me understand how to work with APIs, manage user data, build reusable components, and troubleshoot deployment issues. I also learned how important it is to test features step by step instead of trying to fix everything at once.

**Retrospective:**  
If I improved this project now, I would make the mobile layout cleaner, improve the loading and error messages, and add stronger movie recommendation features.

## Lakeside Coffee Co.

Lakeside Coffee Co. is a static website for a fictional cafe. The goal of this project was to create a clean business website with simple navigation, branding, and useful information for customers.

**Tech Stack:**  
HTML, CSS, JavaScript, GitHub, Deployment

**Project Link:**  
https://shiny-sherbet-7ec23a.netlify.app/contact

**GitHub Repository:**  
https://github.com/Mickey7401/react-static-story

**What I Learned:**  
This project helped me practice basic website structure, visual design, page layout, and responsive styling. It was a good foundation project because it focused on making a site that looks professional and makes sense to a user.

**Retrospective:**  
If I improved this project now, I would add online ordering, better mobile styling, animations, and a more polished menu section.
